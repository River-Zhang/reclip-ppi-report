05

Delta scoring

Compute probability deltas, effect, loss, and gain as mutant minus WT.

Score Definitions

The XGBoost head outputs four class probabilities:proba_0 no effect、proba_1 disrupting、proba_2 increasing、proba_3 decreasing。

delta_proba_c = mutant_proba_c - wt_proba_c

delta_effect = (p1 + p2 + p3)mut - (p1 + p2 + p3)wt

delta_loss = (p1 + p3)mut - (p1 + p3)wt

delta_gain = p2mut - p2wt

How increasing / decreasing is assigned

Each mutant row is compared with the WT baseline for the same pair_id + target position, so every delta is mutant probability minus WT probability.

dominant_effect_class is not the class with the largest mutant probability. It is the class with the largest increase among delta_proba_1, delta_proba_2, and delta_proba_3.

If delta_proba_2 is largest, the mutation is labeled increasing; if delta_proba_3 is largest, it is labeled decreasing; if delta_proba_1 is largest, it is labeled disrupting.

delta_effect only asks whether the mutation becomes less like no effect. Directionality should be read from delta_gain, delta_loss, or the dominant class.

Interpretation Boundary

This model is a mutation-effect classifier, not a standalone binary PPI interaction classifier. The WT baseline pair ranking is useful as an overview of model signal, but should not be interpreted directly as a binary interaction probability. Target and partner are directional because the target protein carries the interest site.

Delta Distribution and Threshold Reference

Overall delta_effect distribution

Teal: most mutant rows below q90. Taller bars mean more rows fall in that delta interval.
Amber: q90 to q95, higher-than-usual signals but not the most extreme tail.
Red: q95 and above, representing the strongest-tail mutation-effect signals.

There is no fixed universal biological threshold for large or small deltas. Here we interpret values relative to the empirical distribution across all mutant rows in this run. Deltas above q95 can be treated as strong signals, and deltas above q99 as extremely strong signals.

delta_effect = mutant_effect - WT_effect, so negative values mean the mutation made the site look closer to no effect. This is expected: the effect probability decreased relative to the WT baseline.

How to Read the Pair Heatmap

Rows are targets and columns are partners. Each cell is a directed target → partner combination; diagonal self cells are excluded.

WT effect baseline: mean(p1+p2+p3) across all WT target positions. Darker cells indicate stronger non-no-effect model signal for that directed pair under WT state; this is not a binary interaction probability.

Top20 mean delta effect: for each directed pair, take the max_delta_effect at each target position, then average the top 20 positions.

Max delta gain: the largest delta_gain, i.e. the mutation with the largest increase in proba_2 increasing. The heatmap is for quick screening and should be read together with the summary, profiles, and top mutation tables below.
